Seletores de atributos CSS

[meu-atributo="valor"] Representa um elemento com um atributo de nome meu-atributo, o qual o valor é exatamente valor.

[meu-atributo1~="valor1"] Representa um elemento com um atributo de nome meu-atributo1, o qual value é uma lista de palavras separadas por espaços, e uma dessas é exatamente valor1.

[meu-atributo2|="value"] Representa um elemento com um atributo de nome meu-atributo2 o qual o valor pode ser exatamente value ou pode começar com value imediatamente seguido por hífen - (U+002D). Isso somente é usado para linguagens que combinam sub-códigos.

[meu-atributo3^="value"] Representa um elemento com um atributo com nome attr que tem um valor prefixado (precedido) por value.

[meu-atributo4$="e"] Representa um elemento com um atributo de nome meu-atributo4 a que tem como sufixo (seguido) por e

[meu-atributo5*="value"] Representa um elemento com um atributo de nome meu-atributo5 o qual valor contém ao menos uma ocorrência de value contido na string.
