Tags HTML

Existem diversas tags HTML para usarmos, entretanto existem diferenças entre elas como:
Tags com display: block e tags com display: in-line
Tags semânticas e tags físicas

Display: block

Tags com esse tipo de display, virarão blocos nas nossas páginas. Assim ocupando um espaço na nossa tela e posteriomente quebrando linha, exemplo: Ao botarmos esse tipo de tag no meio de um texto, o texto não ficara fluido, como era antes.

Display: in-line

Tags com esse tipo de display, serão fluidos nas nossas páginas.

Tags semânticas

Estão relacionadas mais para a estrutura e o significado do conteúdo, além disso contribuem para acessibilidade, SEO e facilitam a leitura para os navegadores.

Tags físicas

São definidas para estilização e layout, são mais genéricas e não fornecem informações sobre a função ou significado do conteúdo.
