Especificidade CSS

A especificação é a maneira de como os navegadores definem quais valores de propriedades são os mais relevantes para o elemento a ser utilizado. A especificação é baseada apenas nas regras impostas na composição de diferentes tipos de seletores.

ID Classe, Pseudo-classe Elemento, Pseudo-elemento
0 1 1
1 0 0
0 2 3

A linha vermelha, contem a maior especificidade das demais linhas.

Entretanto, se houvesse um seletor com a mesma especificidade, a selecionada seria o seletor mais recente

E temos a propiedade !important que possuira maior especificidade
