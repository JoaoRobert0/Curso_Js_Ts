Nessa aula foi abordado sobre Id e Class

Id

Os "id" servem como identificadores para os elementos, entretato os "id" serve para apenas um elemento, ou seja, ele é usado para um elemento "importante".

Para usarmos id, escolhemos uma tag e escrevemos o atributo "id="[nome do identificador]"".
E no css -para criarmos um id- escrevemos "#[nome do id] {escopo do id}".

O atributo id ele possui uma especificidade muita grande, ou seja, ele possui mais importancia, um exemplo: id > class.
Isso quer dizer que o id sobrescrevera a class.

Class

Já as "class" são identificadores porem podem ser usadas por mais de um elemento, ou seja, quando quisermos dar caracteristicas iguais para vários elementos.

Para usarmos class, escolhemos uma tag e escrevemos o atributo "class="[nome do identificador]"".
E no css -para criarmos uma class- escrevemos ".[nome da class] {escopo da class}".
