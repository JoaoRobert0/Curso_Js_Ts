Pseud-elements

Assim como as pseudo-classes, os pseudoelementos em CSS são usados para aplicar estilos a partes específicas de um elemento, mas eles visam criar seletores para partes do conteúdo que não estão presentes no documento HTML. Pseudoelementos são representados por dois pontos (::) e são usados para selecionar e estilizar partes específicas de um elemento que não são representadas diretamente no código HTML.
